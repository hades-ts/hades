# HadesClient

**Bound to**: `HadesClient` as *Singleton*

This is a thin wrapper around `Client` from `discord.js`.

https://discord.js.org/docs/packages/discord.js/14.20.0/Client:Class

It automatically provides the following intents:

- `Guilds`
- `MessageContent`
- `GuildMessages`
- `GuildIntegrations`
- `DirectMessages`
